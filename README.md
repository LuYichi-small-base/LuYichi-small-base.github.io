<font size=5 face="汉仪粗黑简">【创新性作业】——厨房里的物理现象大揭秘</font>

- **<font face="汉仪楷体简">（一）现象观察与记录</font>**

<font face="汉仪楷体简">任务要求：学生在寒假期间，每天至少观察厨房中的一种物理现象，如水的沸腾、油的加热、食物的烹饪过程等，并详细记录下观察到的现象。记录内容包括现象发生的环境（如温度、时间等）、现象的具体表现（如颜色变化、形状变化、声音变化等）以及自己对现象的初步思考。</font>

<font face="汉仪楷体简">以表格形式进行记录，表格内容包括日期、现象名称、现象描述、环境条件、初步思考等栏目。</font>

- **<font face="汉仪楷体简">（二）实验探究与分析</font>**

<font face="汉仪楷体简">任务要求：学生根据观察到的物理现象，选择其中 2 - 3 个自己感兴趣的进行实验探究。设计简单的实验方案，利用厨房中的常见物品作为实验器材，如锅、碗、瓢、盆、筷子、勺子、水、食用油、盐、糖等，验证自己对现象的初步思考是否正确，并进一步分析现象背后的物理原理。</font>

<font face="汉仪楷体简">每个实验探究需撰写一份实验报告，报告内容包括实验目的、实验器材、实验步骤、实验现象、实验结论以及实验过程中的注意事项和可能存在的误差分析。</font>

- **<font face="汉仪楷体简">（三）创意应用与拓展</font>**

<font face="汉仪楷体简">任务要求：学生在完成实验探究后，思考如何将所学的物理知识和实验结果应用到厨房中的实际问题解决中，提出一些创意性的应用方案或改进措施。例如，如何利用水的沸腾原理快速煮熟食物、如何根据油的密度和沸点特点进行食物的油炸等。</font>

<font face="汉仪楷体简">撰写一份创意应用报告，报告内容包括应用背景、应用方案、预期效果以及可能遇到的问题及解决方案。</font>

- **<font face="汉仪楷体简">（四）作业成果展示</font>**

<font face="汉仪楷体简">学生需将**观察记录表、实验报告和创意应用报告**整理成一个完整的作业成果文件，可以采用 `Word` 文档、`PPT` 演示文稿或手写报告的形式。文件中应包含清晰的文字描述、实验数据、图表、照片等，以直观地展示研究过程和成果。</font>
